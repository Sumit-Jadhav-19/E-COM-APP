# E-Commerce Site

A simple e-commerce application built using React, Tailwind CSS, and a dummy API to simulate product listings and shopping functionality.

# üåê Live Demo
You can view the live version of the app here:\
[Live Demo](https://e-com-app-olive.vercel.app/)

## ‚ú® Features
- **View Products**: Browse through various products, view details like price, rating, and more.

- **Product Details**: Each product has its own detail page where users can see more information.

- **Add to Cart**: Add products to your shopping cart.

- **View Cart**: View all the items in your cart, with options to adjust quantity or remove items.

- **Checkout**: Simulate a checkout process where users can see the total price.

- **Responsive Design**: Fully responsive, built using Tailwind CSS.

## üöÄ Technologies Used
- **React**: For building the UI components and managing state.

- **Tailwind CSS**: For styling and creating a responsive layout.

- **Dummy API**: Used to simulate fetching product data (like product names, prices, ratings, etc.).

##üèóÔ∏è Installation
To run the project locally, follow the instructions below:
### Prerequisites
Make sure you have Node.js and npm installed on your system.
### Steps
1. Clone the repository:
   ```
    git clone https://github.com/Sumit-Jadhav-19/E-COM-APP.git
   ```
2. Navigate to the project directory:
   ```
   cd E-COM-APP
   ```
3. Install the dependencies:
   ```
   npm install
   ```
4. Run the development server:
   ```
   npm start
   ```
5. Open your browser and go to http://localhost:3000 to see the app in action.

## üõ† Usage
- **Browse Products**: The homepage shows a variety of products with ratings and prices.

- **Add to Cart**: Click the "Add to Cart" button on any product to add it to your shopping cart.

- **View Cart**: Click on the cart icon to view your cart and proceed to checkout.

- **Checkout**: At the checkout screen, you'll see a summary of your cart items, and the total cost.
